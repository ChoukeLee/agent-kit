# Agent Kit

A lightweight TypeScript framework for safe AI tool-calling. Core principle: **AI understands intent. Code executes. Never let AI directly control state.**

## What this is
An open-source project extracted from [Maison Gourmande](../MG). Every tool has `validate()` (blocks bad calls) + `execute()` (runs only if validate passes). Works with Claude, OpenAI, DeepSeek.
